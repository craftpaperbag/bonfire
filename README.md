# Bonfire 🔥

全ての人が、毎日の活動の拠点となる「ぬくもりのあるリンク集」を簡単に作成・管理できるツールです。

## コンセプト

Bonfireは、特に厳格な社内ネットワークなどの閉鎖的な環境で働く人々が、自分専用のスタートページを迷わず、素早く、そして心地よく構築できるように設計されています。

- **ぬくもりとシンプル**: 暗闇に灯る焚き火のような、落ち着いたダークモードのデザイン。
- **スピーディーな動作**: ローカルHTMLとJSのみで完結し、バックエンドサーバーは不要です。
- **丁寧なガイド**: 日本語UIと言語化されたヘルプにより、誰でも迷わず使えます。
- **安定と安心**: 外部サービスに依存せず、完全にローカルな環境でも動作可能です。

## 主な機能

- **Markdownによる簡単編集**: `Ctrl + B` でエディタを開き、慣れ親しんだマークダウン形式で内容を更新できます。
- **多彩な独自レイアウト**: カード、グリッド、リンクカードなどの独自記法により、情報を整理して表示できます。
- **アイコン & 画像対応**: FontAwesomeのアイコン（`icon:xxx`）や、ローカル画像（`image:xxx.png`）を簡単に挿入可能。
- **リアルタイムプレビュー**: 編集内容は即座に反映され、ブラウザのWebStorageに自動保存されます。
- **永続化支援**: ブラウザに保存されたデータをクリップボードにコピーし、`data.js` ファイルへ反映させるための仕組みを備えています。
- **完全ローカルモード対応**: 外部CDNを使わず、すべてのリソースをオフラインで完結させる手順を内蔵しています。

## 使い方

1. **アプリを開く**: `index.html` をブラウザで開きます。
2. **編集する**: `Ctrl + B` (Macは `Cmd + B`) を押してエディタを開きます。
3. **内容を書き換える**: サンプルデータを見ながら、自分好みの構成に変更します。
4. **データを永続化する**:
   - 画面上部の通知バーにある「データをコピー」ボタン、またはエディタ内の「コピー」ボタンを押します。
   - プロジェクトフォルダにある `data.js` を開き、`bonfireUserData` 変数の内容をコピーしたテキストで上書き保存します。
   - これにより、ブラウザをクリアしてもデータが保持されます。

## 独自記法ガイド

| 機能 | 記述例 | 説明 |
| :--- | :--- | :--- |
| **カード** | `::: card タイトル` <br> 内容 <br> `:::` | 枠囲みのカードを作成します。 |
| **リンクカード** | `::: link [タイトル](URL)` <br> `image:xxx.png` <br> 説明文 <br> `:::` | リッチなリンクカード（バナー型）を作成します。 |
| **グリッド** | `::: grid` <br> 内容1 <br> `\|` <br> 内容2 <br> `:::` | コンテンツを横並びに配置します（`\|` で区切ります）。 |
| **中央寄せ** | `::: center` <br> 内容 <br> `:::` | 中身を中央揃えにします。 |
| **アイコン** | `icon:home` | FontAwesomeのアイコンを表示します。 |
| **画像** | `image:photo.png` | `images/` 内の画像を表示します（角丸）。 |
| **プロフィール画像** | `image@:user.png` | `images/` 内の画像を正円（プロフィール風）で表示します。 |
| **補足テキスト** | `(テキスト)` | 文字を薄く、控えめに表示します。 |

## 完全ローカルモードの設定

インターネットに接続できない環境で使用する場合は、以下の手順で設定してください（詳細はアプリ内の「完全ローカルモードの設定方法」を参照）。

1. 必要なライブラリ（`marked.min.js`, Font AwesomeのCSS/Fonts）をダウンロードし、フォルダ内に配置します。
2. `index.html` 内のCDN読み込みリンクを、ダウンロードしたローカルパスに書き換えます。
